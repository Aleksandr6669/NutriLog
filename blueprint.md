# Blueprint: Calorie Tracker App

## Обзор

Это мобильное приложение для отслеживания калорий, созданное с помощью Flutter. Оно помогает пользователям следить за своим питанием, физической активностью и достигать поставленных целей по здоровью. Приложение имеет чистый, современный дизайн с использованием Material Design 3, кастомных шрифтов и интуитивно понятной навигации.

## Стиль и Дизайн

- **UI Framework**: Flutter с Material Design 3.
- **Основной цвет (Seed Color)**: `Colors.deepPurple`.
- **Шрифты**: `Manrope` от Google Fonts для основного текста, `Oswald` для крупных заголовков.
- **Иконография**: `material_symbols_icons` для четких и современных иконок.
- **Карточки**: Стиль с закругленными углами (`borderRadius: 16`), тонкой обводкой и тенью для эффекта глубины.
- **Кнопки**: Закругленные (`borderRadius: 12`), с основным цветом и жирным текстом.
- **Нижняя навигация**: Кастомная, с эффектом размытия фона (`BackdropFilter`) и плавной анимацией выбора.
- **Тема**: Единая светлая тема. Темная тема была удалена для упрощения.
- **Локализация**: Даты форматируются для русского языка (`ru_RU`).

## Функционал и Экраны

### 1. **Главный экран (`main.dart`)**
   - Инициализирует приложение.
   - Настраивает тему Material 3.
   - Управляет основной навигацией с помощью кастомной нижней панели.

### 2. **Навигация (`MainScreen`)**
   - **Дневник (`HomeScreen`)**: Основной экран для отслеживания приемов пищи, воды и шагов.
   - **Рецепты (`RecipesScreen`)**: Раздел для просмотра и (в будущем) добавления рецептов.
   - **Анализ (`StatsScreen`)**: Экран для просмотра статистики и прогресса.
   - **Профиль (`ProfileScreen`)**: Просмотр и редактирование данных пользователя.

### 3. **Экран "Дневник" (`home_screen.dart`)**
   - Отображает текущую дату с возможностью выбора другой даты.
   - Показывает суммарную информацию по калориям, белкам, жирам и углеводам за день.
   - Имеет секции для завтрака, обеда, ужина и перекусов.
   - Отображает прогресс по шагам и выпитой воде.

### 4. **Экран "Профиль" (`profile_screen.dart`)**
   - Отображает основную информацию о пользователе: аватар и имя.
   - Позволяет сменить аватар, выбрав изображение из галереи.
   - **Карточка "Физические параметры"**: Показывает возраст, рост, вес и пол. Кнопка "Редактировать" ведет на `EditPhysicalParamsScreen`.
   - **Карточка "Дневные цели"**: Показывает цели по весу, калориям, воде, шагам и макронутриентам. Кнопка "Редактировать" ведет на `EditGoalsScreen`.
   - **Карточка "Настройки"**: Содержит переключатель уведомлений.
   - Кнопка "Выйти" для выхода из профиля (функционал не реализован).

### 5. **Редактирование профиля (Разделено на два экрана)**
   - **`edit_physical_params_screen.dart`**: 
     - Позволяет пользователю изменять свое имя, пол, возраст, рост и текущий вес.
     - Содержит форму с валидацией полей.
     - При сохранении обновляет `UserProfile` через `ProfileService` и возвращается на экран профиля.
   - **`edit_goals_screen.dart`**: 
     - Позволяет редактировать целевой вес, дневные нормы калорий, макронутриентов (БЖУ), шагов и воды.
     - Также использует форму с валидацией.
     - При сохранении обновляет `UserProfile` и возвращается на экран профиля.

### 6. **Сервис Профиля (`profile_service.dart`)**
   - **Хранение данных**: Использует `shared_preferences` для сохранения и загрузки данных профиля, что обеспечивает совместимость с мобильными и веб-платформами.
   - **`loadProfile()`**: Асинхронно загружает JSON-строку из `shared_preferences`, декодирует ее и возвращает объект `UserProfile`. Если данные отсутствуют или повреждены, создает и сохраняет профиль по умолчанию.
   - **`saveProfile()`**: Асинхронно кодирует объект `UserProfile` в JSON-строку и сохраняет ее в `shared_preferences`.

### 7. **Модель данных (`user_profile.dart`)**
   - Неизменяемый класс `UserProfile` с методом `copyWith` для удобного обновления полей.
   - Включает поле `avatarImagePath` для хранения пути к аватару.
   - Включает поля для личных данных, целей и истории веса.
   - Содержит методы `fromJson` и `toJson` для сериализации/десериализации.
