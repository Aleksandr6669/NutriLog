# Користувацька угода NutriLog

Дата набрання чинності: 03.05.2026

## 1. Збір та використання даних

NutriLog збирає дані про ваше харчування, вагу, фізичні параметри та цілі, щоб надавати персоналізовані розрахунки КБЖВ та рекомендації. Ваші дані використовуються виключно для функціонування програми та покращення вашого досвіду.

## 2. Зберігання даних

Всі ваші дані (щоденник харчування, рецепти, антропометричні дані) зберігаються безпосередньо в пам'яті програми на вашому пристрої або у вашому особистому обліковому записі NutriLog. Ми не передаємо вашу особисту інформацію третім особам без вашої прямої згоди.

## 3. Використання нейромережевих технологій

Програма використовує сучасні технології нейронних мереж для аналізу ваших прийомів їжі, автоматичного розпізнавання продуктів за описом або фото, а також для допомоги у складанні раціону та створенні рецептів. Обробка даних відбувається анонімно.

## 4. Відповідальність

Програма є інструментом для моніторингу харчування і не замінює консультацію лікаря або професійного дієтолога. Усі розрахунки мають рекомендаційний характер.
